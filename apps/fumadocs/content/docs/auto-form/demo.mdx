### Extraits de code correspondants

#### Schéma simple (article)

```ts
const articleSchema = z.object({
  title: z.string().min(1, "Requis"),
  category: z.enum(["tech", "design", "business"]),
  content: z.string().optional(),
  published: z.boolean().default(false),
});

function ArticleForm() {
  return <AutoForm schema={articleSchema} onSubmit={handleSubmit} />;
}
```

#### Validation avec refine (signup)

```ts
const signupSchema = z.object({
  username: z.string().min(3),
  password: z.string().min(8),
  confirm: z.string(),
}).refine((data) => data.password === data.confirm, {
  message: "Les mots de passe ne correspondent pas",
  path: ["confirm"],
});
```

#### Mode composé avec champs conditionnels

```tsx
function DynamicForm() {
  const [showExtra, setShowExtra] = useState(false);

  return (
    <AutoForm schema={schema} onSubmit={handleSubmit}>
      <div className="grid grid-cols-2 gap-3">
        <AutoField name="title" />
        <AutoField name="category" />
      </div>
      <AutoField name="content" />
      <button type="button" onClick={() => setShowExtra(!showExtra)}>
        {showExtra ? "Masquer" : "Afficher"} les options avancées
      </button>
      {showExtra && <AutoField name="published" />}
      <button type="submit">Publier</button>
    </AutoForm>
  );
}
```
