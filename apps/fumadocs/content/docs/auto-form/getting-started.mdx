## Installation

```bash
npx shadcn@latest add WAROL52/code2-base-ui/auto-form-tanstack-shadcn
```

<Callout type="info">
Le preset `auto-form-tanstack-shadcn` assemble automatiquement le package
cœur, le provider Zod, l'adaptateur TanStack Form et les composants shadcn.
</Callout>

## Quick Start

### 1. Définir un schéma Zod

```ts
import { z } from "zod";

const userSchema = z.object({
  name: z.string().min(2, "Le nom doit faire au moins 2 caractères"),
  email: z.string().email("Email invalide"),
  age: z.number().min(18, "Âge minimum : 18 ans").optional(),
  role: z.enum(["admin", "user", "guest"]),
  active: z.boolean().default(true),
});
```

### 2. Créer le formulaire automatique

```tsx
import { createAutoForm } from "@code2-base-ui/auto-form";
import { zodProvider } from "@code2-base-ui/auto-form-provider-zod";
import { tanstackFormAdapter } from "@code2-base-ui/auto-form-adapter-tanstack";
import { createShadcnRegistry } from "@code2-base-ui/auto-form-render-shadcn";

const { AutoForm, AutoField, useForm, useField } = createAutoForm({
  provider: zodProvider,
  form: tanstackFormAdapter,
  registry: createShadcnRegistry(),
});
```

### 3. Utiliser le preset (encore plus simple)

```ts
// Le preset fait l'assemblage pour vous
import { AutoForm, AutoField } from "~/src/lib/auto-form/preset";
```

### 4. Rendre le formulaire

```tsx
function UserForm() {
  return (
    <AutoForm
      schema={userSchema}
      onSubmit={(data) => console.log("Submitted:", data)}
      className="space-y-4 max-w-md"
    />
  );
}
```

Le formulaire généré automatiquement :
- `name` → `<InputField>` (string)
- `email` → `<InputField type="email">`
- `age` → `<InputField type="number">`
- `role` → `<SelectField>` (enum → options)
- `active` → `<CheckboxField>` (boolean)

## Exemples

### Formulaire simple

```tsx
const schema = z.object({
  title: z.string().min(1, "Requis"),
  description: z.string().max(500).optional(),
  published: z.boolean().default(false),
});

function ArticleForm() {
  return (
    <AutoForm
      schema={schema}
      onSubmit={(data) => fetch("/api/articles", { method: "POST", body: JSON.stringify(data) })}
    />
  );
}
```

### Avec layout personnalisé

```tsx
function TwoColumnLayout() {
  return (
    <AutoForm
      schema={schema}
      className="grid grid-cols-2 gap-4"
    />
  );
}
```

### Mode composé (champs manuels)

Quand vous passez des `children`, `AutoForm` ne génère plus les champs
automatiquement et laisse le contrôle à votre JSX :

```tsx
function CustomForm() {
  return (
    <AutoForm schema={schema} onSubmit={handleSubmit}>
      <div className="grid grid-cols-2 gap-4">
        <AutoField name="name" />
        <AutoField name="email" />
      </div>
      <div className="col-span-2">
        <AutoField name="bio" />
      </div>
      <button type="submit">Envoyer</button>
    </AutoForm>
  );
}
```

### Utilisation directe des composants de champ

```tsx
import { InputField, SelectField, TextareaField } from "@code2-base-ui/auto-form-render-shadcn";

function ContactForm() {
  return (
    <form onSubmit={handleSubmit}>
      <InputField name="name" label="Nom" required />
      <InputField name="email" label="Email" type="email" />
      <SelectField
        name="subject"
        label="Sujet"
        options={[
          { value: "support", label: "Support" },
          { value: "sales", label: "Commercial" },
        ]}
      />
      <TextareaField name="message" label="Message" />
    </form>
  );
}
```

### Validation personnalisée

```tsx
const schema = z.object({
  password: z.string().min(8, "8 caractères minimum"),
  confirm: z.string(),
}).refine((data) => data.password === data.confirm, {
  message: "Les mots de passe ne correspondent pas",
  path: ["confirm"],
});

function SignupForm() {
  return <AutoForm schema={schema} onSubmit={handleSignup} />;
}
```

### Champs conditionnels

```tsx
function DynamicForm() {
  const [showExtra, setShowExtra] = useState(false);

  return (
    <AutoForm schema={baseSchema} onSubmit={handleSubmit}>
      <AutoField name="name" />
      <AutoField name="email" />
      <button type="button" onClick={() => setShowExtra(!showExtra)}>
        {showExtra ? "Masquer" : "Afficher"} les options avancées
      </button>
      {showExtra && (
        <>
          <AutoField name="newsletter" />
          <AutoField name="theme" />
        </>
      )}
      <button type="submit">S'inscrire</button>
    </AutoForm>
  );
}
```
