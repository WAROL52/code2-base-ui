## Définir un schéma

Les formulaires sont générés à partir de JSON Schema :

```ts
const userSchema = {
  type: "object",
  properties: {
    name: { type: "string", title: "Nom", minLength: 2 },
    email: { type: "string", format: "email", title: "Email" },
    age: { type: "integer", title: "Âge", minimum: 18 },
    role: {
      type: "string",
      title: "Rôle",
      enum: ["admin", "user", "guest"],
    },
    bio: { type: "string", title: "Bio", uiWidget: "textarea" },
    active: { type: "boolean", title: "Actif" },
  },
  required: ["name", "email", "role"],
}
```

## Mode simple — `createAutoForm`

Assemble la configuration une fois pour toutes :

```tsx
import { createAutoForm, tanstackAdapter } from "@code2-base-ui/auto-form-builder"
import { createShadcnRegistry } from "@code2-base-ui/auto-form-builder/fields"

const AutoForm = createAutoForm({
  adapter: tanstackAdapter,
  registry: createShadcnRegistry(),
})

function UserForm() {
  return (
    <AutoForm
      schema={userSchema}
      onSubmit={(data) => console.log("Submitted:", data)}
      className="max-w-md space-y-4"
    />
  )
}
```

## Mode render-prop — `AutoFormBuilder`

Pour un contrôle total (layout custom, champs manuels, etc.) :

```tsx
import { AutoFormBuilder, tanstackAdapter } from "@code2-base-ui/auto-form-builder"
import { createShadcnRegistry } from "@code2-base-ui/auto-form-builder/fields"

function CustomForm() {
  return (
    <AutoFormBuilder
      adapter={tanstackAdapter}
      registry={createShadcnRegistry()}
      schema={userSchema}
      onSubmit={(data) => console.log(data)}
    >
      {({ fields, form }) => (
        <form onSubmit={(e) => { e.preventDefault(); form.handleSubmit() }}>
          <div className="grid grid-cols-2 gap-4">
            {fields.map((field) => (
              <AutoFormField
                adapter={tanstackAdapter}
                fieldMeta={field}
                form={form}
                key={field.path}
                registry={createShadcnRegistry()}
              />
            ))}
          </div>
          <button type="submit" className="mt-4 rounded bg-primary px-4 py-2 text-white">
            Envoyer
          </button>
        </form>
      )}
    </AutoFormBuilder>
  )
}
```

## Changer d'adaptateur

Un seul import à changer :

```tsx
import { tanstackAdapter } from "@code2-base-ui/auto-form-builder/adapters/tanstack"
import { rhfAdapter }      from "@code2-base-ui/auto-form-builder/adapters/rhf"
import { formischAdapter } from "@code2-base-ui/auto-form-builder/adapters/formisch"
```

## Layout personnalisé

Surchargez seulement ce dont vous avez besoin :

```tsx
import { createAutoForm, tanstackAdapter, shadcnLayout } from "@code2-base-ui/auto-form-builder"

const AutoForm = createAutoForm({
  adapter: tanstackAdapter,
  registry: createShadcnRegistry(),
  layout: {
    ...shadcnLayout,
    SubmitButton: ({ children }) => (
      <button className="..." type="submit">
        {children ?? "Go!"}
      </button>
    ),
  },
})
```
