title: Installation
description: Installer auto-form-builder dans votre projet
---

## npm

```bash
pnpm add @code2-base-ui/auto-form-builder
```

## Dépendances

Le package a des `peerDependencies` optionnelles — installez seulement
celles dont vous avez besoin :

```bash
# Au moins un form manager (tous optionnels) :
pnpm add @tanstack/react-form   # pour tanstackAdapter
pnpm add react-hook-form        # pour rhfAdapter
pnpm add @formisch/react        # pour formischAdapter
```

## Sous-chemins d'import

```ts
// Principal
import { AutoFormBuilder, createAutoForm, shadcnLayout } from "@code2-base-ui/auto-form-builder"
import { createShadcnRegistry, ShadcnTextField }           from "@code2-base-ui/auto-form-builder/fields"
import { shadcnLayout }                                    from "@code2-base-ui/auto-form-builder/layout"

// Adaptateurs (import direct)
import { tanstackAdapter }  from "@code2-base-ui/auto-form-builder/adapters/tanstack"
import { rhfAdapter }       from "@code2-base-ui/auto-form-builder/adapters/rhf"
import { formischAdapter }  from "@code2-base-ui/auto-form-builder/adapters/formisch"

// Utilitaires
import { mockAdapter }      from "@code2-base-ui/auto-form-builder/testing"
import { createSchemaValidator } from "@code2-base-ui/auto-form-builder/validate"
```
