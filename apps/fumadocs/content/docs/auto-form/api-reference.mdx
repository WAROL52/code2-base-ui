## createAutoForm

```ts
function createAutoForm(config: CreateAutoFormConfig): AutoFormSystem
```

Assemble les trois couches (provider, form, registry) en un système complet
de formulaire automatique.

### CreateAutoFormConfig

```ts
interface CreateAutoFormConfig {
  provider: SchemaProviderFactory;
  form: FormStateAdapter;
  registry: FieldRegistry;
  layout?: LayoutStrategy;
}
```

### AutoFormSystem

```ts
interface AutoFormSystem {
  AutoForm: <TSchema>(props: AutoFormProps<TSchema> & { children?: ReactNode }) => ReactNode;
  AutoField: (props: { name: string }) => ReactNode;
  useForm: () => FormAPI;
  useField: (name: string) => FieldController;
  AutoFormProvider: typeof AutoFormProvider;
}
```

**AutoForm** — Composant racine qui génère un formulaire complet à partir
d'un schéma.

**AutoField** — Composant qui rend un champ individuel identifié par son nom.

**useForm** — Hook retournant l'API du formulaire (`values`, `errors`,
`submit`, `reset`, `dirty`, `isSubmitting`).

**useField** — Hook retournant le contrôleur d'un champ (`value`, `onChange`,
`onBlur`, `error`, `touched`).

## AutoFormProvider

```ts
function AutoFormProvider(props: {
  children: ReactNode;
  form: FormAPI;
  fields?: Record<string, FieldController>;
}): ReactNode
```

Contexte React qui rend les hooks `useFormContext` et `useFieldContext`
accessibles dans l'arborescence.

---

## useFormContext / useFieldContext

```ts
function useFormContext(): FormAPI | null
function useFieldContext(): FieldController | null
```

Hooks de contexte pour accéder à l'API du formulaire ou au contrôleur
d'un champ depuis n'importe quel descendant de `AutoFormProvider`.

---

## FieldProvider

```ts
function FieldProvider(props: {
  children: ReactNode;
  field: FieldController;
}): ReactNode
```

Wrapper contextuel pour exposer un `FieldController` via `FieldContext`.

## Types du package cœur

```ts
interface SchemaProvider<TSchema = unknown> {
  readonly fields: FieldMeta[];
  readonly jsonSchema: JsonSchema;
  validate: (data: unknown) => ValidationResult;
  getFieldMeta: (path: string) => FieldMeta | undefined;
}

interface SchemaProviderFactory {
  readonly name: string;
  create: <T>(schema: T) => SchemaProvider<T>;
}

interface FormStateAdapter {
  readonly name: string;
  useForm: (config: {
    defaultValues?: Record<string, unknown>;
    validate: (data: unknown) => ValidationResult;
  }) => FormAPI;
  useField: (name: string) => FieldController;
}

interface FormAPI {
  values: Record<string, unknown>;
  errors: Record<string, string | undefined>;
  submit: (e: { preventDefault: () => void }) => void;
  reset: () => void;
  dirty: boolean;
  isSubmitting: boolean;
}

interface FieldController {
  value: unknown;
  onChange: (value: unknown) => void;
  onBlur: () => void;
  error?: string;
  touched: boolean;
}

interface AutoFormProps<TSchema = unknown> {
  schema: TSchema;
  onSubmit?: (data: unknown) => void | Promise<void>;
  defaultValues?: Record<string, unknown>;
  children?: ReactNode;
  className?: string;
}

interface LayoutStrategy {
  readonly name: string;
  render: (
    fields: FieldMeta[],
    renderField: (field: FieldMeta) => ReactNode,
  ) => ReactNode;
}
```

## zodProvider

```ts
import { zodProvider } from "@code2-base-ui/auto-form-provider-zod";

const provider: SchemaProviderFactory = {
  name: "zod",
  create: <T>(schema: T) => SchemaProvider<T>;
};
```

SchemaProviderFactory pour Zod. Traduit un schéma Zod (`z.object({...})`)
en métadonnées `FieldMeta[]` via `flatfields` et assure la validation
avec `safeParse`.

### ZodProvider

```ts
class ZodProvider<TSchema extends z.ZodType> implements SchemaProvider<TSchema> {
  readonly fields: FieldMeta[];
  readonly jsonSchema: JsonSchema;
  constructor(schema: TSchema);
  validate(data: unknown): ValidationResult;
  getFieldMeta(path: string): FieldMeta | undefined;
}
```

## createShadcnRegistry

```ts
import { createShadcnRegistry } from "@code2-base-ui/auto-form-render-shadcn";

function createShadcnRegistry(): FieldRegistry
```

Crée un `FieldRegistry` préconfiguré avec les composants shadcn/ui :

| Type / Format | Composant |
|---|---|
| `string` | InputField |
| `string.format = email \| url \| tel \| password` | InputField |
| `string.widget = textarea` | TextareaField |
| `string.widget = select` | SelectField |
| `string.widget = radio` | RadioGroupField |
| `string.widget = otp` | OtpField |
| `string.format = date \| date-time` | DatePickerField |
| `number` | InputField |
| `number.widget = select` | SelectField |
| `number.widget = slider` | SliderField |
| `boolean` | CheckboxField |
| `boolean.widget = switch` | SwitchField |
| `object.widget = collapse` | FieldGroup |
| `array` | ArrayField |
